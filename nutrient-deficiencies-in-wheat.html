Nutrient Deficiencies in Wheat

Overview:

Our team explored a 1200 image corn data set and 420 image wheat data set. We used Teachable Machine to achieve a model with 97% accuracy in detecting nitrogen deficient wheat and 94% accuracy in detecting healthy nitrogen level wheat. Additional data analysis was completed using Python and Tableau.

The end goal is a website where users can upload wheat images for analysis by the model and return results of healthy or deficient levels of nitrogen

Technologies Used:

  • Tools: Teachable Machine
  • Data Analysis: Python, Tableu
