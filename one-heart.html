One Heart: A One-Stop Digital Platform

Overview:

The One Heart platform is a transformed version of the current Heart for Good nonprofit organization. My team and I's mission in this remodel was three-fold: recruitment, outreach, and retention. In order to reach these goals, we refreshed the website, created a hidden admin portal, and built an integrated reminder system.

Website Refresh

The main issues we addressed with the previous website was visual clutter and busyness. We updated all the pages to have a more sleek and streamlined design. Additionally, we created a registration form, for both volunteers and participants, as well as program ambassadors.

Centralized Portal Development

One of the largest issues we found with the previous set up was that there was not a centralized platform where the administration could see everything. To combat this, we created a portal where admin can easily acess website views, donations, create new events, admin-to-admin messaging, notifications and more.

Moreover, this portal handles the event reminder information. The reminder system sends out emails and text messages to event volunteers and participants within two weeks of their registered event. To add, it handles alert notifications for people that recieve stage 2 or 3 diagnoses from the EKG screenings.

Technologies Used:

  • Frontend: React, HTML, CSS, Material UI
  • Backend: Firebase Database, Node.js, Google Maps API

Affiliation:

This project was part of the JP Morgan Chase Code for Good Hackathon 2024. Our team took first place, winning the Hackathon for the nonprofit organization Heart in the Game.
